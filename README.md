# 🌌 NeuroSphere

> *Not just what you say, but how you feel.*

A futuristic **AI-powered mental wellness web application** that blends emotional intelligence, real-time conversation, and cyberpunk aesthetics into one seamless platform.

---

## ✨ What is NeuroSphere?

NeuroSphere is more than just another chat interface. It's an **emotionally-aware AI companion** designed to:

- 🧠 **Understand** your emotional state in real-time
- 💬 **Adapt** its responses based on your mood patterns
- 📊 **Track** your emotional wellness journey
- 🎯 **Generate** personalized wellness insights
- 🎤 **Listen** via voice input for hands-free interaction

Built with a **cyberpunk-inspired UI**, smooth animations, and intelligent context-aware responses, NeuroSphere transforms mental wellness into a modern, engaging experience.

## 🚀 Getting Started

### Prerequisites

```bash
Node.js (v16+)
npm or yarn
Modern browser with Web Speech API support
```

### Installation

```bash
# Clone the repository
git clone https://github.com/khyatiagrawal-2025/neurosphere.git

# Navigate to project directory
cd neurosphere

# Install dependencies
npm install

# Start development server
npm run dev
```

The application will be available at `http://localhost:5173`
